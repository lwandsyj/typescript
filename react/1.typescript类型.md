1. 原始类型
   
   + string: 字符串
   + number：数字
   + boolean: 布尔类型
   + undefined: 未定义
   + null: 

2. 其他类型：
   + enum : 枚举
   + array: Type[] 或者Array< Type >
   + any: 类型类型，相当于JavaScript，可以赋值任意类型
   + 自定义class,type,interface 
   + 联合类型： 可以赋值多种类型
   + function: 函数类型
        
            (param1,....paramn)=>void(或者Type)   （参数）=>(返回) void(Type)(函数返回值类型)
   + void： 没有返回值

3. 类型定义使用  :Type 语法
   
        let a: number ;// number 类型

4. 类型定义可以用于变量，函数参数，函数返回值。
5. 对象定义使用interface 或者 Type
