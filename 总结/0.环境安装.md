1. 安装package.json 
   
        npm init -y 
2. 安装typescript
   
        npm i typescript -S
3. 使用ts-node 执行ts 文件
   
        npm i ts-node -g

        ts-node index.ts  // 运行index.ts

4. 使用tsc 编译ts 文件
   
+ tsc 是安装typescript 自带的
  
        tsc index.ts
5. 使用tsc 创建tsconfig.json
   
+ tsconfig.json ts 编译配置文件
        
        tsc --init
